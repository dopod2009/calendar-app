## 📥 安装指南

### 系统要求

- **最低版本**: Android 8.0 (API 26)
- **推荐版本**: Android 13+ (API 33+)
- **存储空间**: 至少50MB可用空间
- **网络**: 需要网络连接

### 安装步骤

1. **下载APK**
   - 从邮件附件下载 `app-release.apk`
   - 或从测试平台下载

2. **启用未知来源**
   ```
   设置 → 安全 → 启用"未知来源应用安装"
   ```

3. **安装应用**
   - 点击APK文件
   - 点击"安装"
   - 等待安装完成

4. **打开应用**
   - 点击"打开"或从桌面启动

5. **授予权限**
   - 允许网络访问
   - 允许通知（Android 13+）

### C. 常见问题

**Q: APK安装失败怎么办？**  
A: 检查是否启用了"未知来源应用安装"权限

**Q: 应用启动很慢怎么办？**  
A: 首次启动需要初始化数据库，稍慢属于正常现象

**Q: 无法登录怎么办？**  
A: 检查网络连接，确认账号密码正确，或使用测试账号

**Q: 提醒不生效怎么办？**  
A: 检查是否授予通知权限，检查系统勿扰模式
